As metas ajudam você a juntar dinheiro para algo específico — uma reserva de emergência, uma viagem, uma entrada — definindo um alvo e acompanhando seu progresso ao longo do tempo.

![Metas](/screenshots/goals.png)

## Criando uma Meta

Vá até **Metas** na barra lateral e clique em **Adicionar Meta**. Preencha:

1. **Nome**: para o que você está economizando
2. **Valor Alvo**: quanto você quer alcançar
3. **Moeda**: a moeda da meta
4. **Data Alvo** (opcional): quando você quer alcançá-la
5. **Tipo de Acompanhamento**: como o progresso é medido (veja abaixo)
6. **Ícone e Cor**: uma identidade visual para a meta

## Tipos de Acompanhamento

O **tipo de acompanhamento** define como o Securo sabe quanto você já economizou:

| Tipo | Como o progresso é medido |
|------|---------------------------|
| **Manual** | Você informa o valor atual e atualiza conforme avança |
| **Conta Vinculada** | O progresso acompanha o saldo de uma conta específica |
| **Ativo Vinculado** | O progresso acompanha o valor de um ativo específico |
| **Patrimônio Líquido** | O progresso acompanha o seu patrimônio líquido total |

O acompanhamento vinculado mantém suas metas atualizadas automaticamente — conforme a conta, o ativo ou o patrimônio mudam, o progresso da meta atualiza junto.

## Progresso e Status

Cada meta mostra uma barra de progresso e um percentual em relação ao alvo. Quando uma meta tem data alvo, o Securo também mostra um status de **no ritmo** para você saber se está mantendo o passo:

| Status | Significado |
|--------|-------------|
| **Adiantado** | Você está economizando mais rápido do que o necessário para bater o alvo no prazo |
| **No Ritmo** | Você está no ritmo de bater o alvo na data alvo |
| **Atrasado** | Você precisa economizar mais rápido para bater o alvo no prazo |
| **Em Atraso** | A data alvo já passou e a meta não foi concluída |
| **Concluída** | A meta foi alcançada |

## Gerenciando Metas

Cada meta tem ações para o seu ciclo de vida:

- **Pausar / Retomar**: interrompa o acompanhamento de uma meta temporariamente e retome depois
- **Marcar como Concluída**: marque uma meta como alcançada
- **Arquivar**: tire uma meta finalizada ou abandonada da lista ativa
- **Editar / Excluir**: altere os detalhes ou remova a meta

Use os filtros **Ativas**, **Concluídas**, **Pausadas**, **Arquivadas** e **Todas** no topo da página para alternar entre elas.

## Metas no Painel

As metas ativas também aparecem em um widget de **Progresso das Metas** no [Painel](/docs-pt/features/dashboard), com uma barra compacta de progresso e o status de ritmo para cada uma — para você ver onde está sempre que abrir o Securo.
