Payees represent the merchants, companies, or people you transact with. Linking transactions to payees gives you better visibility into where your money goes, beyond just categories.

![Payees](/screenshots/payees.png)

## Payee Types

Securo supports two payee types:

- **Company**: Businesses and organizations (e.g., Amazon Prime)
- **Merchant**: Stores, restaurants, and shops (e.g., Starbucks)

## Viewing Payees

Navigate to **Payees** in the sidebar to see a searchable table of all your payees. Each row shows:

- **Favorite star**: Starred payees appear at the top
- **Name**: The payee's display name
- **Description**: Optional notes about the payee
- **Type**: Company or Merchant
- **Transactions**: How many transactions are linked to this payee
- **Edit**: Click to modify the payee

## Adding a Payee

1. Click **+ Add Payee**
2. Enter the payee name
3. Add an optional description
4. Select the type (Company or Merchant)
5. Click **Save**

## Using Payees with Rules

Payees work hand-in-hand with [rules](/docs/features/rules). You can create rules that automatically assign a payee to transactions based on their description. For example:

- Rule: Description contains "STARBUCKS" → Set payee to "Starbucks"
- Rule: Description starts with "AMAZON" → Set payee to "Amazon Prime"

This means you don't have to manually tag every transaction, rules handle it for you.

## Searching Payees

Use the search bar at the top of the payees page to quickly find a specific payee by name. This is useful when you have many payees and need to edit one or check its transaction count.
