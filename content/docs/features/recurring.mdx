Recurring transactions let you define subscriptions and regular bills that repeat on a schedule. Instead of manually adding the same transaction each month, Securo can generate them for you automatically.

![Recurring Transactions](/screenshots/recurring.png)

## How It Works

A recurring transaction is a template that defines:

- **Description**: What the payment is for (e.g., "Spotify", "Netflix USD")
- **Amount**: The transaction value, including currency
- **Frequency**: How often it repeats (monthly, weekly, etc.)
- **Next date**: When the next occurrence is due
- **Status**: Active or paused

When the scheduled date arrives, Securo creates an actual transaction in the appropriate account based on the template.

## Creating a Recurring Transaction

1. Navigate to **Recurring** in the sidebar
2. Click **+ Add Recurring**
3. Enter the description
4. Set the amount and currency
5. Choose the account
6. Select the frequency (monthly, weekly, bi-weekly, etc.)
7. Set the start date
8. Click **Save**

## Managing Recurring Transactions

The recurring transactions page shows a table with all your schedules:

| Column     | Description                                    |
| ---------- | ---------------------------------------------- |
| Description | The name of the recurring payment             |
| Amount     | The value and currency (with converted amount) |
| Frequency  | How often it repeats                           |
| Next Date  | When the next transaction will be generated    |
| Status     | Active or paused                               |
| Actions    | Edit or delete the schedule                    |

## Generating Pending Transactions

Click **Generate Pending** to create all overdue recurring transactions at once. This is useful if you haven't opened Securo in a few days and need to catch up.

## Multi-Currency Subscriptions

For subscriptions in foreign currencies (e.g., a USD subscription on a BRL account), Securo shows both the original amount and the converted equivalent. This makes it easy to see the real cost of international subscriptions.

## Use Cases

- **Streaming services**: Netflix, Spotify, Disney+
- **Software subscriptions**: Cloud services, SaaS tools
- **Rent/mortgage**: Regular housing payments
- **Insurance premiums**: Monthly or annual insurance
- **Loan payments**: Regular debt repayments
