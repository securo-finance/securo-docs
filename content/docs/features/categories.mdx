Categories are how Securo organizes your spending. Every transaction can be assigned to a category, and those categories power your budgets, reports, and dashboard widgets. Securo uses a **two-level hierarchy**: category groups contain individual categories.

![Categories](/screenshots/categories.png)

## Category Groups

Category groups are the top-level organizers. Securo comes with sensible defaults:

- **Housing**: Rent, mortgage, home expenses
- **Food**: Groceries, dining out, delivery
- **Transportation**: Gas, public transit, rideshare
- **Lifestyle**: Health, entertainment, education, personal care
- **Other**: Everything else

Each group has its own icon and can be expanded or collapsed in the category list.

## Individual Categories

Inside each group, you'll find specific categories. For example, the Food group might contain:

- **Dining**: Restaurants and dining out
- **Groceries**: Grocery stores and supermarkets

Each category has:
- A **name**
- An **icon** (emoji-based)
- A **color** (hex code) for use in charts and visualizations
- A link to its **parent group**

## Creating a Category

1. Go to **Categories** in the sidebar
2. Click **+ Add Category**
3. Enter the category name
4. Choose an icon and color
5. Select the parent group
6. Click **Save**

## Creating a Category Group

1. On the Categories page, click **+ Add Group**
2. Enter the group name
3. Choose an icon
4. Click **Save**

New categories can then be added inside this group.

## Editing Categories

Click the pencil icon next to any category or group to edit its name, icon, color, or parent group. You can also reorganize categories by moving them between groups.

Use the **Collapse All** toggle at the top to quickly collapse or expand all groups.

## How Categories Are Used

Categories flow through the entire application:

- **Transactions**: Each transaction is assigned to one category
- **Budgets**: You set spending limits per category
- **Dashboard**: The spending chart breaks down expenses by category
- **Reports**: All analytics are grouped by category
- **Rules**: Automation rules can auto-assign categories based on transaction descriptions

Uncategorized transactions are highlighted on the dashboard so you're always aware of what needs attention.
