O Securo é um aplicativo de finanças pessoais self-hosted que te dá controle total dos seus dados financeiros. Acompanhe várias contas em diferentes moedas, categorize transações automaticamente com regras, defina orçamentos e monitore seu patrimônio líquido, tudo a partir de uma interface limpa e moderna.

![Painel do Securo](/screenshots/dashboard.png)

## Por que Securo?

Gerenciar finanças pessoais não deveria exigir entregar seus dados para terceiros. O Securo roda na sua própria infraestrutura, então suas informações financeiras nunca saem do seu servidor. Ele suporta **múltiplas moedas** com conversão automática, o que o torna ideal para quem movimenta dinheiro entre países.

Diferente de planilhas ou trackers básicos, o Securo oferece um conjunto completo de ferramentas que trabalham juntas:

- **Transações** fluem para **categorias** através de **regras** que você define
- **Orçamentos** acompanham seus gastos contra os limites por categoria a cada mês
- **Transações recorrentes** cuidam de assinaturas e contas regulares automaticamente
- **Relatórios** e **acompanhamento de patrimônio líquido** mostram o panorama ao longo do tempo

## Por onde começar?

Se você é novo no Securo, recomendamos este caminho:

1. Siga o guia de [Instalação](/pt-BR/docs/getting-started/installation) para colocar o app no ar
2. Passe pelo [Assistente de Configuração](/pt-BR/docs/getting-started/setup-wizard) para criar sua primeira conta
3. Leia a visão geral do [Painel](/pt-BR/docs/features/dashboard) para entender a tela inicial
4. Configure [Categorias](/pt-BR/docs/features/categories) e [Regras](/pt-BR/docs/features/rules) para automatizar a classificação de transações

## Stack Tecnológica

| Camada    | Tecnologia        |
| --------- | ----------------- |
| Backend   | FastAPI (Python)  |
| Frontend  | React + Vite      |
| Banco     | PostgreSQL        |
| Deploy    | Docker Compose    |
