---
title: Visão Geral dos Recursos
description: Explore todos os recursos do Securo para gerenciar suas finanças pessoais.
---

O Securo oferece um conjunto completo de ferramentas para acompanhar e gerenciar suas finanças pessoais. Cada recurso foi pensado para funcionar em conjunto: as transações fluem para categorias por meio de regras, os orçamentos acompanham os gastos por categoria e os relatórios mostram o panorama completo.
