Os orçamentos ajudam a controlar gastos definindo um limite mensal para cada categoria. Conforme você gasta, o Securo acompanha quanto do orçamento foi usado e quanto ainda resta, te dando uma visão clara se você está dentro do esperado.

![Orçamentos](/screenshots/budgets.png)

## Como Funcionam os Orçamentos

Um orçamento no Securo é simples: você escolhe uma categoria e define o valor máximo que quer gastar por mês com ela. O Securo então compara seu gasto real nessa categoria com o limite.

Por exemplo, se você orçar R$500 para Mercado e já gastou R$320 neste mês, a visão de orçamento mostra que ainda restam R$180.

## Criando um Orçamento

1. Navegue até **Orçamentos** na barra lateral
2. Clique em **+ Adicionar Orçamento**
3. Selecione uma categoria no menu suspenso
4. Informe o valor do limite mensal
5. Clique em **Salvar**

Você pode criar orçamentos para quantas categorias quiser. Cada orçamento funciona de forma independente.

## Visão Mensal

A página de orçamentos mostra o mês atual por padrão, com setas de navegação para percorrer meses anteriores. Para cada orçamento você vê:

- O **nome** e o **ícone** da categoria
- O valor **orçado** (seu limite)
- O valor **gasto** até agora
- O valor **restante**
- Uma **barra de progresso** mostrando o quão perto você está do limite

Quando você ultrapassa um orçamento, a barra de progresso e o valor restante ficam vermelhos para te avisar.

## Dicas para Orçar com Eficiência

- **Comece pelas categorias maiores**: orce primeiro as áreas onde você mais gasta (moradia, alimentação, transporte)
- **Revise mensalmente**: ao fim de cada mês, veja quais orçamentos ficaram apertados ou folgados demais e ajuste
- **Use junto com regras**: configure [regras de categorização](/pt-BR/docs/features/rules) para que as transações sejam categorizadas automaticamente, mantendo o acompanhamento preciso sem trabalho manual
- **Confira o painel**: o widget de gastos por categoria no painel dá uma rápida verificação da saúde do orçamento sem precisar abrir a página de orçamentos
