# Securo Docs

Documentation site for [Securo](https://github.com/securo-finance/securo), an open-source personal finance manager.

Built with [Fumadocs](https://fumadocs.dev) and [Next.js](https://nextjs.org).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Adding Screenshots

1. Drop your PNG/JPG into `public/screenshots/`
2. Reference in MDX: `![Alt text](/screenshots/filename.png)`

## Deployment

Push to GitHub and connect to [Vercel](https://vercel.com) for automatic deployments.
